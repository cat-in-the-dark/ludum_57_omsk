# Махо-сэдзе битва за титул королевы триангуляции

2025-04-05 Ludum Dare 57

![demo gif](screenshots/demo_short.gif)

A classic sport game of Magic Girls is the Tennis with Tentacles! You have a big BAT to swing to send back big balls to your enemy.

### Controls

- WASD or joystick for Movement
- Space or B to Swing the Bat
- ESC or menu button to Restar
